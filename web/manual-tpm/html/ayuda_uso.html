Características Técnicas

Esta guía fue desarrollada en HTML, lo cual indica que puede ser visualizado como una página web común.

Condiciones Recomendadas

Para una mejor visualización de esta guía, verifique que:

  1. Navegador recomendado Firefox Versión 2 o superior.
    • Su navegador de internet soporte instrucciones "JavaScript".
  2. La resolución de vídeo sea de: 1024 X 768. (Recomendado)
  3. La calidad del color sea de 32 bits.
  4. Monitor a Color.

Si tiene problemas en visualizar esta guía adecuadamente o desconoce la manera de constatar si su equipo cumple con las condiciones recomendadas, consulte la ayuda de un experto.
